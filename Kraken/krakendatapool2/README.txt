Para ejecutar las pruebas se realizaron los siguientes archivos con el siguiente paso a paso:
- Ejecutar el comando "node index.js" el cual creará los archivos properties, teniendo en cuenta el archivo .JSON MOCK_DATA.json.
- Una vez con esos archivos properties generados, se ejecuta el script de bash (consola cmd de windows), y este ejecutara n veces los test de kraken (n cantidad de filas archivo MOCK_DATA.json).
- El archivo .feature señala el escenario base para cada uno de los escenarios que se ejecutarán.
